# 📱 LAB-15 — SQLite et Android

## 📌 Description

Ce projet Android démontre l’utilisation de **SQLite** pour gérer une base de données locale dans une application mobile.
Il permet d’effectuer les العمليات الأساسية (CRUD) :

* ➕ Ajouter des données
* 📖 Lire des données
* ✏️ Modifier des données
* ❌ Supprimer des données

---

## 🎯 Objectifs

* Comprendre le fonctionnement de **SQLite** sur Android
* Manipuler une base de données locale
* Implémenter les opérations CRUD
* Structurer un projet Android propre

---

## 🛠️ Technologies utilisées

* **Java**
* **Android Studio**
* **SQLite**
* **XML (UI)**

---

## 📂 Structure du projet

```
app/
 ├── java/                # Code source Java
 ├── res/                 # Layouts XML (interface)
 ├── AndroidManifest.xml  # Configuration de l'application
```

---

## 🚀 Fonctionnalités

* Création automatique de la base de données
* Gestion des enregistrements (CRUD)
* Interface simple pour interagir avec les données

---

## ▶️ Exécution du projet

1. Cloner le projet :

```bash
git clone https://github.com/hichamdrt/LAB-15-SQLite-et-Android.git
```

2. Ouvrir avec **Android Studio**

3. Lancer l’application sur :

* 📱 Émulateur
* 📲 Appareil réel
